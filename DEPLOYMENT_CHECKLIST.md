# 本地部署检查清单

## 📋 部署前准备

### 环境检查

- [ ] **Node.js 已安装**
  - 版本要求: v16+
  - 验证命令: `node -v`
  - 推荐版本: v18 LTS 或 v20 LTS

- [ ] **npm 已安装**
  - 验证命令: `npm -v`
  - 应该随Node.js自动安装

- [ ] **微信开发者工具 已安装**
  - 下载地址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
  - 登录微信账号
  - 确认可以正常打开

- [ ] **HBuilderX 已安装** (可选但推荐)
  - 下载地址: https://www.dcloud.io/hbuilderx.html
  - 选择 App开发版或标准版

### 项目文件检查

- [ ] **项目代码已下载**
  - 从Git克隆: `git clone <项目地址>`
  - 或下载ZIP并解压

- [ ] **项目文件完整**
  - [ ] package.json
  - [ ] manifest.json
  - [ ] pages.json (在src目录)
  - [ ] tsconfig.json
  - [ ] vite.config.ts
  - [ ] README.md

## 🚀 安装步骤

### 依赖安装

- [ ] **进入项目目录**
  ```bash
  cd student-planner-miniapp
  ```

- [ ] **安装依赖**
  ```bash
  npm install
  ```
  - 如果失败，尝试: `npm install --registry=https://registry.npmmirror.com`
  - 等待安装完成（可能需要几分钟）

- [ ] **验证安装**
  - [ ] node_modules 目录已创建
  - [ ] package-lock.json 文件已创建

### 项目配置

- [ ] **配置小程序AppID** (可选)
  - 打开 `manifest.json`
  - 找到 `mp-weixin.appid`
  - 填入你的小程序AppID
  - 或留空使用测试号

- [ ] **配置日志系统** (可选)
  - 打开 `src/config/logging.config.json`
  - 根据需要调整日志级别
  - 默认级别: INFO

## ▶️ 启动项目

### 使用启动脚本（推荐）

**Windows:**
- [ ] 双击 `start.bat`
- [ ] 等待依赖检查和安装
- [ ] 等待开发服务器启动
- [ ] 看到 "编译完成" 提示

**Mac/Linux:**
- [ ] 给脚本添加执行权限: `chmod +x start.sh`
- [ ] 运行脚本: `./start.sh`
- [ ] 等待依赖检查和安装
- [ ] 等待开发服务器启动
- [ ] 看到 "编译完成" 提示

### 手动启动

- [ ] **运行开发服务器**
  ```bash
  npm run dev:mp-weixin
  ```

- [ ] **等待编译完成**
  - 看到 "Build complete" 或类似提示
  - 检查 `dist/dev/mp-weixin` 目录是否创建

### 用微信开发者工具打开

- [ ] 打开微信开发者工具
- [ ] 点击 `+` 或 `导入项目`
- [ ] 选择项目目录: `dist/dev/mp-weixin`
- [ ] 填入项目信息:
  - [ ] 项目名称: 大学生规划
  - [ ] AppID: 你的AppID或留空
  - [ ] 开发模式: 小程序
- [ ] 点击 `导入`
- [ ] 等待项目加载
