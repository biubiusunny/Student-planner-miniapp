# 本地部署和测试指南

## 📋 前置要求

### 必需软件

1. **Node.js** (v16+)
   - 下载地址: https://nodejs.org/
   - 推荐: v18 LTS 或 v20 LTS
   - 验证安装: `node -v` 和 `npm -v`

2. **HBuilderX** (推荐)
   - 下载地址: https://www.dcloud.io/hbuilderx.html
   - 选择: App开发版 (包含App开发)
   - 或: 标准版 (仅小程序)

3. **微信开发者工具** (必须)
   - 下载地址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 用于预览和调试微信小程序

### 可选软件

- Git: 用于版本控制 (https://git-scm.com/)
- VSCode: 推荐的代码编辑器 (https://code.visualstudio.com/)

## 🚀 快速开始

### 方式1: 使用HBuilderX (推荐)

#### 步骤1: 下载项目代码

```bash
# 如果有Git
git clone <项目地址>
cd student-planner-miniapp

# 或者直接下载ZIP压缩包并解压
```

#### 步骤2: 用HBuilderX打开项目

1. 打开 HBuilderX
2. 点击 `文件` → `打开目录`
3. 选择 `student-planner-miniapp` 文件夹
4. 等待项目加载完成

#### 步骤3: 配置小程序AppID

1. 在项目根目录找到 `manifest.json`
2. 双击打开，选择 `微信小程序配置`
3. 在 `AppID` 输入框填入你的小程序AppID
4. 如果没有AppID，可以填测试号AppID或留空

**获取小程序AppID:**
1. 登录微信公众平台: https://mp.weixin.qq.com/
2. 注册小程序账号 (如果没有)
3. 在 `开发` → `开发管理` → `开发设置` 中查看AppID

#### 步骤4: 安装依赖

在HBuilderX中:
1. 右键点击项目根目录
2. 选择 `使用命令行窗口打开所在目录`
3. 执行命令:

```bash
npm install
```

等待依赖安装完成（可能需要几分钟）

#### 步骤5: 运行项目

在HBuilderX中:
1. 点击顶部菜单 `运行`
2. 选择 `运行到小程序模拟器` → `微信开发者工具`
3. 首次运行会要求选择微信开发者工具的安装路径
4. HBuilderX会自动编译并打开微信开发者工具

#### 步骤6: 测试功能

在微信开发者工具中:

**基础功能测试:**
1. ✅ 页面导航是否正常
2. ✅ 创建课程是否成功
3. ✅ 创建任务是否成功
4. ✅ 创建目标是否成功
5. ✅ 专注模式是否正常
6. ✅ GPA计算是否正确
7. ✅ 数据导入导出是否正常

**日志监控测试:**
1. 点击 `我的` → `日志监控`
2. 查看是否有日志记录
3. 尝试筛选和搜索功能
4. 测试导出和清空功能

## 🔧 常见问题

### 问题1: npm install 失败

**原因:** 网络问题或npm源问题

**解决方案:**

```bash
# 方案1: 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com

# 方案2: 使用cnpm
npm install -g cnpm --registry=https://registry.npmmirror.com
cnpm install

# 方案3: 使用yarn
npm install -g yarn
yarn install
```

### 问题2: HBuilderX无法识别项目

**原因:** 项目缺少必要的配置文件

**解决方案:**

确保以下文件存在:
- `manifest.json` ✓
- `pages.json` ✓
- `package.json` ✓
- `tsconfig.json` ✓
- `vite.config.ts` ✓

### 问题3: 微信开发者工具打开失败

**原因:** 编译失败或路径错误

**解决方案:**

1. 检查编译日志，查看是否有错误
2. 确保 `dist/dev/mp-weixin` 目录存在
3. 手动用微信开发者工具打开该目录
4. 清除编译缓存: `运行` → `清理运行缓存`

### 问题4: 运行时白屏

**原因:** 代码错误或依赖问题

**解决方案:**

1. 打开微信开发者工具的控制台查看错误信息
2. 检查 `src/pages.json` 中的页面路径是否正确
3. 确保所有依赖都已安装
4. 尝试重新编译: `运行` → `重新运行`

### 问题5: 日志监控显示异常

**原因:** logger工具未正确初始化

**解决方案:**

1. 检查 `src/utils/logger.ts` 是否存在
2. 确认 `src/App.vue` 中已导入logger
3. 在浏览器控制台查看是否有错误
4. 尝试清空日志后重新记录

1. 在微信开发者工具中打开 `调试器`

3. 找到对应的源文件
4. 点击行号设置断点
5. 触发相应操作
6. 查看变量值

### 3. 网络请求调试

1. 打开 `调试器` → `Network`
2. 执行网络操作
3. 查看请求详情
4. 检查响应数据
