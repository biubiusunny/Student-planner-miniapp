## 快速开始

### 1. 克隆项目
```bash
cd /workspace/projects/workspace/student-planner-miniapp
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置 AppID
在 `manifest.json` 中填写小程序 AppID：
```json
{
  "mp-weixin": {
    "appid": "你的AppID"
  }
}
```

### 4. 配置订阅消息模板ID
在 `src/utils/notification.ts` 中填写：
```typescript
export const TEMPLATE_IDS = {
  COURSE_REMINDER: '课程提醒模板ID',
  TASK_REMINDER: '任务提醒模板ID',
  GOAL_PROGRESS: '目标进度模板ID'
}
```

### 5. 运行项目
使用 HBuilderX：
- 打开项目
- 点击 运行 → 运行到小程序模拟器 → 微信开发者工具

使用命令行：
```bash
npm run dev:mp-weixin
```

## 常见问题

### Q1: 图标如何制作？
A: 可以使用在线工具如 [iconfont](https://www.iconfont.cn/)、[Figma](https://www.figma.com/) 制作，或找设计师。

### Q2: 白噪音音频从哪里获取？
A: 可以从免费音频网站下载，如 Freesound、Pixabay Audio，或自己录制。

### Q3: 订阅消息模板审核需要多久？
A: 通常 1-3 个工作日，建议提前申请。

### Q4: 如何测试订阅消息？
A: 在微信开发者工具的"订阅消息"模块中测试，或在真机上测试。

### Q5: 数据存储有限制吗？
A: 单个 key 最多 10MB，总存储 10MB，注意定期清理旧数据。
