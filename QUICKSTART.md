# 快速入门指南

## 🎯 5分钟快速部署

### 步骤1: 准备环境（2分钟）

**必需软件：**

1. **Node.js** (v16+)
   - 下载: https://nodejs.org/
   - 安装后验证: 打开命令行输入 `node -v`

2. **微信开发者工具** (必须)
   - 下载: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 用于预览和调试小程序

3. **HBuilderX** (推荐)
   - 下载: https://www.dcloud.io/hbuilderx.html
   - 选择 App开发版或标准版

### 步骤2: 下载项目（1分钟）

**方式A: 使用Git**
```bash
git clone <项目地址>
cd student-planner-miniapp
```

**方式B: 直接下载**
- 下载ZIP压缩包
- 解压到任意目录

### 步骤3: 启动项目（2分钟）

**最简单的方式：使用启动脚本**

**Windows:**
```bash
双击 start.bat
```

**Mac/Linux:**
```bash
chmod +x start.sh
./start.sh
```

**或使用命令行：**
```bash
npm install
npm run dev:mp-weixin
```

### 步骤4: 用微信开发者工具打开

1. 打开微信开发者工具
2. 点击 `+` 或 `导入项目`
3. 项目目录选择: `dist/dev/mp-weixin`
4. 填入小程序AppID:
   - 测试时可以留空
   - 正式发布需要申请AppID
5. 点击 `导入`

✅ 完成！现在可以看到小程序了！

## 🐛 遇到问题？

### 常见问题

**Q: npm install 失败？**

A: 使用淘宝镜像
```bash
npm install --registry=https://registry.npmmirror.com
```

**Q: HBuilderX无法识别项目？**

A: 确保文件完整，检查这些文件是否存在:
- manifest.json
- pages.json
- package.json

**Q: 微信开发者工具打开失败？**

A:
1. 检查 `dist/dev/mp-weixin` 目录是否存在
2. 尝试手动用微信开发者工具打开该目录
3. 清除编译缓存: `运行` → `清理运行缓存`

**Q: 页面白屏？**

A:
1. 打开微信开发者工具控制台查看错误
2. 检查网络连接
3. 尝试重新编译

**Q: 日志监控没有记录？**

A:
1. 确认 `src/utils/logger.ts` 存在
2. 检查 `src/App.vue` 是否导入了logger
3. 尝试执行一些操作生成日志
